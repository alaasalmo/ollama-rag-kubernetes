Building the chunks

1- Divide the text to chunk --> Embedding --> Save to ChromaDB
2- Query --> embedding --> find the chunk in ChromaDB --> Junks become context --> Send it to OLLAMA

Vectrtor embedding & Similarity search

                         ┌─────────────────────┐
                         │ Raw Documents (txt, │
                         │ PDF, HTML, etc.)    │
                         └─────────┬───────────┘
                                   │
                                   ▼
                       ┌────────────────────────┐
                       │ Split into Chunks      │
                       │ (e.g. by paragraphs)   │
                       └─────────┬──────────────┘
                                 │
                                 ▼
                    ┌────────────────────────────┐
                    │ Convert Chunks to Vectors  │
                    │ (use embedding model)      │
                    └──────────┬─────────────────┘
                               │
                               ▼
                    ┌────────────────────────────┐
                    │ Store in ChromaDB          │
                    │ (embedding + metadata)     │
                    └──────────┬─────────────────┘
                               ▼
                    ┌────────────────────────────┐
                    │ ChromaDB is now ready      │
                    │ for query-time retrieval   │
                    └────────────────────────────┘



                        ┌──────────────────── ┐
                        │   User Question     │
                        │ e.g., "What is RAG?"│
                        └─────────┬────────── ┘
                                  │
                                  ▼
                   ┌────────────────────────────┐
                   │ Embed the Query (Vector)   │
                   │ Same model used as storage │
                   └─────────┬──────────────────┘
                             │
                             ▼
                  ┌──────────────────────────────┐
                  │ Search ChromaDB Vector Store │
                  │ (find top-k relevant chunks) │
                  └─────────┬────────────────────┘
                            │
                            ▼
                  ┌──────────────────────────────┐
                  │  Retrieved Document Chunks   │
                  └─────────┬────────────────────┘
                            │
                            ▼
                  ┌──────────────────────────────┐
                  │ Build Final Prompt           │
                  │ (Context + Query + Optional  │
                  │ Instructions)                │
                  └─────────┬────────────────────┘
                            ▼
                   ┌───────────────────────────┐
                   │ Call Ollama Model (LLM)   │
                   └────────────┬──────────────┘
                                ▼
                      ┌────────────────────┐
                      │   Final Answer     │
                      └────────────────────┘
